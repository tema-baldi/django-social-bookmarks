Веб-приложение для сохранения и обмена изображениями с социальными функциями — подписки, лайки, лента активности. Проект построен на основе книги «Django 4 в примерах» (Антонио Меле).
## Основные возможности

- ✅ Аутентификация — регистрация, вход/выход, смена и сброс пароля
- ✅ Профили пользователей — дата рождения, аватар, редактирование.
- ✅ Социальная аутентификация — вход через Google (OAuth 2.0).
- ✅ Букмарклет — добавление изображений с любого сайта одним кликом.
- ✅ Система подписок — следить за другими пользователями и видеть их активность.
- ✅ Лента активности — действия пользователей (лайки, закладки, подписки) с обобщёнными связями.
- ✅ Подсчёт просмотров изображений и рейтинг популярных изображений (с использованием Redis).
